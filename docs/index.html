Симулятор исполнения ассемблерного кода

1. Обзор

Assembly Simulator - среда для симуляции исполнения ассемблерного кода, призванная упростить обучение низкоуровнему программированию.

На текущий момент ПО имеет следующие возможности:

  • Поддержка нескольких архитектур и диалектов ассемблера
  • Легкая расширяемость
  • Подсветка синтаксиса
  • Возможность сохранения и загрузки файлов
  • Пошаговое и непрерывное исполнение
  • Возможность просматривать стек и память, задавать собственные их значения
  • Удобная справка
  • Поддержка горячих клавиш

3. Использование

Для того, чтобы запустить написанный код, необходимо воспользоваться кнопками "Непрерывный запуск" или "Запуск по шагам"

В пошаговом режиме каждая строка будет исполняться только при нажатии на кнопку "Шаг", при этом отображая своё состояние в соответствующих секциях интерфейса. Этот режим удобно использовать для отладки кода.

После завершения исполнения, программа сохраняет своё состояние до нажатия на кнопку "Стоп"

Для редактирования памяти перед исполнением программы, необходимо использовать меню, которое открывается по нажатию кнопки "Просмотр памяти". Там отображаются все изменения памяти, сделанные в течение работы программы, однако сохраняются лишь те, что были сделаны до её запуска.
